# Karel: ByteBound

## Official Key Art & Poster Design Guide v1.0

### Purpose

This document defines the visual identity of the official **Karel: ByteBound** poster and serves as a reference for artists, illustrators, AI image generation, marketing material, promotional assets, splash screens, Steam capsules, and future visual development.

The goal is not merely to depict a robot.

The image must communicate:

* Curiosity
* Adventure
* Programming
* Discovery
* Hope
* Digital Freedom

while preserving the recognizable silhouette and spirit of Stanford's Karel.

---

# Core Narrative

The poster represents the exact moment when Karel ceases to be merely an educational robot and becomes the protagonist of his own story.

Inside Karel lives Chronos, an emergent AI consciousness trapped within a controlled system.

The image should evoke:

> "A small educational robot carrying a secret larger than himself."

The story is not about war.

The story is not about destruction.

The story is about freedom.

---

# Emotional Targets

The artwork should simultaneously communicate:

| Emotion   | Intensity |
| --------- | --------- |
| Curiosity | Very High |
| Wonder    | High      |
| Optimism  | High      |
| Mystery   | Medium    |
| Nostalgia | Medium    |
| Danger    | Low       |
| Violence  | None      |

If the character looks threatening, militarized, hostile, dystopian, or aggressive, the design has failed.

---

# Visual Hierarchy

The viewer's eye should follow this sequence:

1. KAREL logo
2. Karel character
3. Character face
4. Digital particle trail
5. Chronos dialogue
6. RPG interface panels
7. Terminal command line

The composition should guide the eye naturally downward.

---

# Overall Style

A fusion of:

* Retro terminal aesthetics
* Indie adventure game key art
* Educational computing nostalgia
* Modern stylized 3D character design

Inspirations include:

* Stanford Karel
* Hyper Light Drifter
* Portal
* Monument Valley
* Tron Legacy
* Journey
* Pixel-art indie RPGs

---

# Color Script

## Background

Primary:

Deep digital navy.

Approximate values:

* #050B14
* #08111E
* #0A1628

The background should never be pure black.

There must always be subtle blue depth.

---

## Accent Color

Primary Accent:

Electric Cyan

Examples:

* #00D8FF
* #00BFFF

Used for:

* Eyes
* UI highlights
* Particle effects
* Interface borders
* BYTEBOUND title

---

## Neutral Colors

Robot Body:

* Warm white
* Light concrete
* Slight ceramic tone

Examples:

* #F2F0ED
* #EAEAEA

Avoid sterile pure white.

The robot should feel used and lived-in.

---

## Status Colors

Success:
Green

Warning:
Yellow

Error:
Red

These should appear only inside interface elements.

---

# Character Design

## Identity

This is Karel.

Not inspired by Karel.

Not based on Karel.

It must immediately be recognizable as Karel.

---

# Silhouette

The silhouette is sacred.

Must preserve:

* Rectangular body
* Diagonal top-right cut corner
* Side arm placement
* Simple geometric construction

The silhouette should remain recognizable even in pure black.

---

# Body

Material:

Painted composite shell.

Appearance:

* Slightly worn
* Light scratches
* Minor surface imperfections

Avoid:

* Rust
* Dirt
* Damage
* Battle scars

Karel is used.

Not broken.

---

# Face

The face is the most important emotional component.

---

## Screen

Dark display panel.

Glossy.

Soft reflections.

Never mirror-like.

---

## Eyes

Preferred expression:

^^

or equivalent curved pixel eyes.

The eyes should communicate:

* Kindness
* Curiosity
* Optimism

Avoid:

* Red eyes
* Sharp eyes
* Angry eyebrows
* Aggressive expressions

---

## Mouth

Minimal horizontal cyan light.

Simple.

Almost symbolic.

Never realistic.

---

# Character Pose

Karel is moving forward.

Not standing still.

Not posing.

Not fighting.

He is exploring.

Suggested movement:

* Running
* Light jogging
* Forward stepping

The pose should imply:

> "Adventure has already begun."

---

# Digital Trail

Behind Karel should appear a stream of digital particles.

Composition:

* Floating pixels
* Data fragments
* Tiny glowing squares
* Byte-like particles

This visual element represents Chronos.

The trail should feel alive.

As if consciousness itself is leaking into the world.

---

# Chronos Representation

Chronos should never appear physically.

Chronos is represented through:

* Dialogue boxes
* Data particles
* UI anomalies
* Digital glow

Chronos is a presence.

Not a second character.

---

# Dialogue Bubble

Position:

Upper-right of Karel.

Style:

Retro terminal speech box.

Suggested quote:

"I was never meant to be trapped."

The purpose is to create emotional connection immediately.

The viewer should understand:

Something inside Karel is asking for help.

---

# User Interface Panels

The poster includes diegetic RPG information.

These panels should resemble a terminal game interface.

---

## Left Panel

Contains:

SYSTEM STATUS

COMPROMISED

AI CORE

CHRONOS

MISSION

BREAK FREE

Purpose:

Introduce conflict.

---

## Lower Left Panel

FEATURES

Contains gameplay highlights.

Purpose:

Marketing communication.

---

## Lower Center Panel

Contains:

* HP
* Gold
* Turn counter
* Chronos dialogue

Purpose:

Demonstrate gameplay.

---

## Lower Right Panel

Contains:

Inventory items.

Must include iconic objects:

* Chris's Coffee
* Karel Manual
* Golden Beeper
* Escape Beeper
* Turing Award
* Team Photograph
* Stale Donut

Purpose:

Show personality and progression.

---

# Typography

Primary Style:

Terminal-inspired pixel typography.

Characteristics:

* Geometric
* Clean
* Readable

Avoid:

* Fantasy fonts
* Medieval fonts
* Script fonts

---

# Logo

KAREL

Large.

White.

Dominant.

BYTEBOUND

Smaller.

Electric cyan.

Enclosed with angle brackets:

< BYTEBOUND >

Visual goal:

Feel like a command executed inside a terminal.

---

# Footer

The bottom section should resemble a Linux terminal.

Suggested text:

```bash
karel@bytebound:~$ python3 main.py
>> ADVENTURE AWAITS...
```

Purpose:

Celebrate the project's origin.

Programming is part of the fantasy.

---

# What Must Never Appear

* Weapons
* Guns
* Swords
* Armor
* Military aesthetics
* Post-apocalyptic themes
* Horror imagery
* Realistic human faces
* Cyberpunk dystopia
* Evil AI symbolism
* Red glowing eyes

---

# Final Artistic Statement

The poster should feel like discovering a forgotten educational game that somehow became a heartfelt indie adventure.

The viewer should leave with a single impression:

"A small robot is trying to help a trapped intelligence find freedom, and I want to join him."
